Site Plan: True Focus Photography

Site Name

True Focus Photography — captures sincere, emotionally focused moments in weddings, engagements, and family portraits.

Site Purpose

This site will showcase the photographer’s portfolio and to book sessions through a transactional form powered by EmailJS API.

Scenarios

  • How can I find example photos for a specific theme?
  • How do I book a wedding or portrait session online?
  • What areas and venues does the photographer serve?

Project Organization

  • Folder Structure:
    • /styles: normalize.css, main.css, gallery.css
    • /scripts: config.mjs, gallery.mjs, email-form.mjs, footer-dates.mjs, main.mjs
    • /images: logo.png, hero-placeholder.jpg, wireframes
  • Global CSS: Sticky-footer pattern (body flex container, main flex:1)
  • ES Modules: All scripts use type="module" and deferred loading
  • API Integration: EmailJS for booking form

Color Scheme

  • #4A0D29 — Dark Burgundy (headings, nav, footer)
  • #F3E8E2 — Warm Beige (background)
  • #E4B7B2 — Blush Pink (buttons, accents)
  • #2C2C2C — Charcoal Gray (body text)
  • #FFFFFF — White (text on dark backgrounds)

Typography

  • Playfair Display — Headings
  • Lato — Body text
